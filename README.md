# Sales & Business Insights Dashboard (Power BI)

This project is an interactive **Sales Analytics Dashboard** built using **Power BI** to analyze business performance across categories, regions, and customer segments.

---

## Project Objective

To create a professional dashboard that provides insights into:

- Total Sales and Profit performance  
- Category-wise revenue contribution  
- Regional profit distribution  
- Monthly sales trends  
- Segment-wise customer sales share  

---

## Tools & Technologies Used

- Power BI  
- DAX Measures  
- Data Cleaning & Transformation (Power Query)  
- Superstore Sales Dataset  

---

## Key KPIs

- Total Sales  
- Total Profit  
- Total Orders  
- Profit Margin %

---

## Dashboard Features

- Sales Trend Analysis (Month-Year)  
- Category-wise Sales Breakdown  
- Profit by Region with Conditional Formatting  
- Segment Contribution using Treemap  
- Interactive Filters (Region, Category, Year)
